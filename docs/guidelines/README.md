This zip contains implementation-ready guidance for coding agents building the FarseerTech website.

## Files

- `farseertech-website-guidelines.md` — Full brand, design, UX, IA, article, SEO, Astro, and implementation guidelines.
- `agent-implementation-brief.md` — Short coding-agent prompt / operating brief.
- `icons.md` — SVG icon style rules and starter icon set.

## Core constraint

Avoid excessive scrolling. The homepage should be compact, high-signal, and mostly fit within 1–2 desktop viewports. Use section readers, drawers, tabs, lightboxes, compact cards, and command search instead of long vertical pages.
